# Realtime Database

Realtime database is used to store and retrieve the recipes as json objects.
Four different endpoints exists:

  - `/beverages`: to store and retrieve "beverages"
  - `/desserts`: to store and retrieve "desserts"
  - `/meals`: to store and retrieve "meals"
  - `/metadata`: to store and retrieve the metadata
    - `/beverages`
    - `/desserts`
    - `/meals`
  
# Storage

Storage is used to store the images associated to the recipes. An URL for the resource is automatically created when the recipes is first created.

  - `/beverages`: to store and retrieve "beverages"
  - `/desserts`: to store and retrieve "desserts"
  - `/meals`: to store and retrieve "meals"
